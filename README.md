# Book Scraper & Data Pipeline

Scrapes book data (title, price, rating) from books.toscrape.com,
cleans it with Pandas, and saves to CSV automatically.

## Tech Stack
Python | Requests | BeautifulSoup4 | Pandas

## Features
- Scrapes multiple pages automatically
- Filters books by price
- Sorts by rating
- Saves all + filtered data to CSV

## How to Run
pip install requests beautifulsoup4 pandas
python book_scraper.py
